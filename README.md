# Event Management System 
Built by Vandana (Star Team)

## Purpose
Automates event creation, ticketing, payments, reminders, and session scheduling.

## Setup Instructions
1. Clone the repo:
   git clone https://github.com/<username>/star-team_event-management-system.git

2. Open folder and run:
   npm install   (if frontend)
   pip install -r requirements.txt (if backend/python)

3. Run application:
   npm start OR python app.py
